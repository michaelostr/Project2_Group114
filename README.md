# Project 2 — Health & Lifestyle Sort
## Group 114 | Safia Hodge, Michael Ostrowski, Luke Dunham
### Requirements
- CLion
- C++17
- SFML 3.0

### Setup

1. Clone the repo and open the project folder in CLion
2. CLion will auto-detect CMakeLists.txt — reload CMake if prompted
3. Place the dataset CSV, fonts, and images in the project root folder
4. Click Run (Shift + F10)
### Usage
- Type “Enter” to continue or type “Exit” to exit
- Select a category from the menu to benchmark merge sort vs quick sort
- Search using the format: <value> in <category>
- Use the left/right arrow keys to browse multiple results
